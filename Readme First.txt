Avant de commencer:
    l'objectif de ce Ytrack v2 est de vous faire apprendre des principes de programmation basiques.
    Le tout avec un projet précis en finalité.


Projet final:
    Créer un jeux au format console (CMD) de vôtre choix (RPG, Clicker, Stratégie ...)
    vous apprendrez petit a petit de nouvelles choses vous permettant de poffiner le jeux.


Système d'exercices:
    Vous aurez un dossier contenant un énoncé et un lien 
    du cours a tour of go pour vous aiguiller. vous écrirez ensuite vôtre fonction puis demanderais une correction.

ATTENTION: Toujours lire les consignes en entier!
